# Guardians of the Ancient Kingdom - Versión GUI

**Autor:** Kevin Tapia Suarez  
**Lenguaje:** Python 3  
**Modo:** Interfaz Gráfica (Tkinter)

## Descripción

Versión gráfica del juego de batalla por turnos "Guardians of the Ancient Kingdom". Los jugadores pueden seleccionar personajes y atacar por turnos mediante botones.

## Instrucciones

1. Ejecuta el archivo `juego_gui.py`.
2. Selecciona un personaje para el Jugador 1.
3. Selecciona un personaje para el Jugador 2.
4. Pulsa "Atacar" para que los personajes se enfrenten hasta que uno pierda toda su vida.

## Clases y habilidades

- **Guerrero:** +20% de daño.
- **Mago:** Ignora defensa enemiga.
- **Arquero:** Doble daño si su ataque > defensa rival.

## Requisitos

- Python 3.8 o superior
- Tkinter (incluido por defecto en Python)

## Ejecución

```bash
python juego_gui.py
